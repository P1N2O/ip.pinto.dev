# ip-pinto-dev
A simple HTTP server that returns the visitor's IP address in Plain Text, JSON, or JSONP and XML format.

## Install dependencies
```bash
bun install
```

## Run Dev Server
```bash
bun run dev
```

## Run Production Server (Bun)
```bash
bun run start
```

## Run Production Server (Docker)
```bash
docker compose up -d --build
```
